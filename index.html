Como usar.


Instale Chisele via Bower

$ bower install chisele --save

Insira plataform.js no html.

Importe Chisele

Por fim use a tag

Opções

Local

Você pode usar a opcão local para passar uma simples array de dados para o autocomplete.

Remote

Use a opção remote para passar uma url para pegar os dados.

* Ao usar o remote o local e automaticamente desativado

* Chisele usa somente JSON.

Root

Se o JSON tiver um elemento Root ele deve ser definido.

Filds

Para mostrar as sugestões e nessesario escolher um elemento neste caso é name

* Você pode adicionar mais de um fild e so usar "," para separar.
