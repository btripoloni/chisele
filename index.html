Chisele

A simple autocomplete web component.

How to use.

Install Chisele by Bower

$ bower install chisele --save

Insert plataform.js in the html.

<script src="//cdnjs.cloudflare.com/ajax/libs/polymer/0.3.4/platform.js"></script>

Import Chisele

<link rel="import" href="bower_components/chisele/chisele.html" >

Use the tag

<chi-sele> </chi-sele>

Options

Local

You can use the option local for to pass a simple array.

<chi-sele local="['Brazil','Germany','Japan']"> </chi-sele>

Remote

Use the option remote to pass a url for fetch the data.

<chi-sele remote="http://urltofetch.com/autocomplete/" > </chi-sele>

* Using remote the local option is automaticaly disable

* Chisele use only JSON.

Root

If the JSON has a Root element he should to be defined.

<chi-sele remote="http://urltofetch.com/autocomplete/" root="results" > </chi-sele>

Filds

To show the suggestions and choose a necessary element in this case is name

<chi-sele remote="http://urltofetch.com/autocomplete/" root="results" filds="name" > </chi-sele> * You can add more than one fild just use "," to separate.

Using CSS

#chisele::shadow input { width: 100%; padding: 10px; border: 2px solid #000; font-family: arial; font-size: 20px; } #chisele::shadow input:focus { border: 2px solid rgba(216, 0, 255, 0.69); } #chisele::shadow #chiseleResults { background-color: #fff; position: absolute; color: #000; width: 95%; padding-top: 5px; padding-bottom: 5px; } #chisele::shadow #chiseleResults p { padding: 3px; margin: 0px; font-size: 15px; } #chisele::shadow #chiseleResults p:hover { background-color: #00a8d1; padding: 3px; }

Using with Javascript

Use QuerySelector

var chisele = document.queryselector('#chisele'); chisele.sValue;//returns the text on the input.
